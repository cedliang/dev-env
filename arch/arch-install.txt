1. Install arch with the packages in arch-packages.txt
2. Use visudo to give wheel sudo privilege
3. Run arch-setup.sh
4. Move .zshrc and .nanorc files into home
5. Install further stuff (ghcup, vscode, chrome, PragmataPro font, etc)

note: if using a desktop env, to change the font in gnome-terminal, install (with pacman) dconf-editor, then /org/gnome/terminal/legacy/profiles:/
